En sockets hay una variable llamada rooms la cual es un array.
rooms = [];
Dentro de rooms estan los objetos que son las salas.
El objeto contiene los siguientes datos; 
    - name (string)
    - isPublic (boolean)
    - mode (string)
    - admin (string)
    - users (Array de strings)
    - id (Number)
    - accessCode (String)
    - accesible (boolean)
